# GECM Lost & Found Management System

## Description

A Spring Boot web application developed for managing lost and found items in a college.

## Features

✔ User Registration
✔ Login
✔ Report Lost Item
✔ Report Found Item
✔ Claim Item
✔ Admin Dashboard
✔ Image Upload
✔ MySQL Database

## Technologies

- Java 17
- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf
- Bootstrap
- Maven



## Installation

1. Clone repository
2. Create MySQL database
3. Import SQL file
4. Update application.properties
5. Run application
